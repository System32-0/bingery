# Bingery
 
> [!NOTE]
> This is a personal project built for personal use and convenience. It is not intended to fully represent my skills or abilities as a developer.
 
> [!IMPORTANT]
> Approximately 80% of this codebase is AI-generated. It was built this way intentionally for efficiency and convenience.
 
A self-hosted media tracking app for anime, manga, manhwa, manhua, books, and movies.

**Live site:** [bingery.cdespalo.com](https://bingery.cdespalo.com)
 
## Features
 
- Track media across six types: Anime, Manga, Manhwa, Manhua, Book, Movie
- Statuses: Planned, In Progress, Completed, Dropped
- Filter and sort your library by type, status, rating, and more
- Quick-add via URL — paste a link from MyAnimeList, AniList, Crunchyroll, Kitsu, or MangaDex to auto-fill metadata
- Detail modal with full entry view and edit/delete controls
- Client-side AES encryption for localStorage data
- Themed settings panel with custom accent color support
- Fully offline-capable after first load — no external dependencies at runtime
